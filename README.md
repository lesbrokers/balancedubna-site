# Balance Dubna Site

Next.js site for balancedubna.ru

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel
- Import this repository into Vercel
- Add domain: balancedubna.ru and www.balancedubna.ru
- Configure DNS in nic.ru exactly as Vercel shows
